# GPT 4 Chat Bot
---
# under maintence
--- 
This is a simple Discord chat bot built using discord.js and the gpt - 4 model from Open AI. This is the same model used for the popular chatbot Chat GPT.


## How to setup

1. Clone the repository to the current directory

```powershell



https://github.com/NICK-FURY-6023/Chat-GPT-4-Bot.git
```

2. Install all the dependencies

- Using npm
```powershell
npm install
```

- Using yarn
```powershell
yarn
```

3. fill your bot token or openai_api

5. Start your bot

- Using npm
```powershell
npm run start
```

- Using yarn
```powershell
yarn start
```
